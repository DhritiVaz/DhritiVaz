### Projects

**[Krypt — Secure Password Manager](https://github.com/DhritiVaz/krypt)**
Python · Flask · PostgreSQL · AES-256-GCM · Argon2id · JWT · TOTP · pytest · Docker

Self-hosted password manager with a zero-knowledge architecture — the server never stores the encryption key. Every vault entry is encrypted with AES-256-GCM using a unique IV, with the key derived at runtime from the master password via Argon2id. Features JWT authentication with refresh token rotation, TOTP two-factor authentication, rate limiting, and append-only audit logging. 96 tests, 93% coverage — unit, integration, and security tests covering SQL injection, auth bypass, and cross-user data isolation.

**[Expense Tracker](https://github.com/DhritiVaz/expense-tracker)** · [Live](https://expense-tracker-ev7t.onrender.com)
Python · Flask · PostgreSQL · SQLAlchemy · JavaScript · Chart.js

Full-stack multi-user expense tracker deployed on Render. Features natural language input parsing ("lunch 80 today"), a smart savings planner that analyses spending patterns and recommends category-level cuts, personality-based insights, streak tracking, and an interactive 4-tab dashboard with Chart.js visualisations.

**[Student Dashboard & Management System](https://github.com/DhritiVaz/student-dashboard)**
React.js · Node.js · Express.js · PostgreSQL · Prisma ORM

Full-stack student management system with admin authentication, RESTful APIs, and a modular React frontend. Converted static HTML into a dynamic SPA with routing, reusable components, and full frontend-backend integration.

**[Centralised Billing & Inventory System](https://github.com/DhritiVaz/centralised-billing-system)**
Node.js · PostgreSQL

Backend system for billing operations and inventory tracking — invoice generation, product management workflows, and PostgreSQL data storage.
